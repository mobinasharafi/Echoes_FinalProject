# Echoes

Core features:
- Role-based auth (JWT): users, authorised reps, moderators
- Case submission (structured + safety checks)
- Dual-track interaction: Leads & Information / Support & Compassion
- Location-based browsing by city/region
- Moderation layer (blacklist + review queue)
- Case-specific private inbox (no personal information)

Tech:
- React (frontend)
- Node.js + Express (backend)
- MongoDB (database)
- Hosted on Vercel (frontend) + (backend host to be decided)
